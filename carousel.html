Introductie

Om mijn development leeruitkomsten aan te tonen had ik wat POC's nodig. Deze heb ik dan ook een aantal gemaakt. Een van mijn projectjes is een carousel. Deze carousel is gemaakt doormiddel van CSS en een stukje javascript.

Carousel


Uitwerking & Bugs

Ik heb dit carousel gebruikt om foto's in op te slaan. Dit is makkelijk om veel foto's bij een bepaald onderwerp te zetten zonder dat ze over de hele pagina gaan staan. Doormiddel van de knoppen aan de randen van het carousel kun je naar de volgende foto, of je kunt terug door de knop links. Er staan ook puntjes onderaan het carousel. Dit is ook een manier van navigeren. Ik heb dit carousel met behulp van W3Schools gemaakt.

Code


Uitleg

De functie showSlides zorgt voor het slideshow effect. Wanneer er op een van de pijltes knoppen of een dan de dots onderin gedrukt wordt gaat het carousel naar de volgende foto.

Zoals je ziet wordt in het CSS bestand het carousel gestijld, het carousel past zich zelf aan op de groote van de foto die er in komt. Ik heb een maximum breedte ingesteld zo dat hij niet te groot wordt. Verder heb ik voor mezelf extra commetaar gezet bij het CSS gedeelte.

Wat heb ik geleerd?

Ik heb geleerd hoe ik een carousel in mijn website plaats en hoe ik deze zo kan aanpassen dat het voor mij het beste uit komt. In het kort, het customizen van dit carousel is wat ik op verschillende pagina's gedaan heb.




Wat doe ik de volgende keer anders?

Ik wil de volgende keer graag een ander carousel gebruiken of iets anders dat functioneert als een carousel, zo leer ik telkens nieuwe technieken en kan ik deze wellicht toepassen in andere projecten.
