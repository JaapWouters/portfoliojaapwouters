Fotoshoot

Voor het project Ruby Mus hebben wij als groep zijnde een fotoshoot geplanned met Myrthe. Ik en Thom zijn voor deze shoot naar Tilburg gegaan.

Stijl

Ik heb de foto's naast het station van Tilburg gemaakt, hier was een soort industrieel terein. Er hingen hier veel jongeren rond. Voor de stijl van de foto's ging ik voor een street/industrieele vibe. Dit omdat ze haar kleding stijl omschrijft als een street stijl. Ik heb sommige foto's ook een lichter tintje gegeven, dit heb ik gedaan omdat sommige foto's anders te donker werden.

3D Photography

Op de 15e slide zie je een gifje van een effect wat ik gecreeërd heb, op deze foto krijg je een soort 3D effect. Doordat ik 2 foto's gemaakt heb op de zelfde hoogte maar vanaf een andere hoek was het mogelijk om dit effect te creeëren. Ik heb dit effect gemaakt met de tijdlijn in Photoshop.

Wat heb ik geleerd?

Ik heb met deze fotoshoot geleerd veel verschillende hoeken te proberen en andere locaties uit te testen. Een andere set kleren bij een bepaald shot uitproberen was ook een leermoment voor mij.

Wat doe ik de volgende keer anders?

De volgende keer zou ik de kleding stijl meer matchen bij de locatie. Bij een 3D effect zou ik de persoon als enige onderwerp in de foto laten staan (geen andere grote objecten in de foto).

Feedback

Myrthe gaf aan tijdens de shoot de foto's al echt dope te vinden. Toen we ons werk presenteerde werd dit nogmaals bevestigd.
